## 进制的换算

| 转换               | 思路                              | 本质                                          |
| ------------------ | --------------------------------- | --------------------------------------------- |
| 十进制 -> 十六进制 | 不断除以 16，记录余数，逆序排列。 | 把十进制的数拆分成以 16 为底的权重组合。      |
| 十六进制 -> 十进制 | 每一位乘以对应的 16 的幂次。      | 一个数的每一位，都代表着不同数量的“16 的幂”。 |
|                    |                                   |                                             |
